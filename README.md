🔥 **APEX FiveM – Stash System (FREE)**

نظام خزنة جديد لسيرفرات FiveM.

**المميزات:**
• خزنة محمية برقم سري
• UI بسيط ونظيف
• يدعم QBCore
• قابل للتعديل من config

**طريقة التثبيت:**

1. حمل السكربت من GitHub.
2. ضع فولدر السكربت داخل مجلد `resources`.
3. أضف السطر التالي في `server.cfg`:

```
ensure apex-stash
```

📄 **شرح إضافة الأيتم موجود داخل ملف:**
